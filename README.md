Food Ordering App

🛠️ Installation & Setup

Clone the repo:

git clone https://github.com/Faith-254-UI/food-ordering-app.git
cd food-ordering-app


Install dependencies:

npm install


Start development server:

npm run dev


Open in browser:

http://localhost:5173
